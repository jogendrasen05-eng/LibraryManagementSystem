&#x20; 📚 Library Management System



A desktop-based Library Management System developed using \*\*Java Swing\*\*, \*\*JDBC\*\*, and \*\*Oracle Database\*\*. This application helps librarians efficiently manage books, members, book issuing, and returning operations.



\---



&#x20;  🚀 Features



\- 🔐 Login System

\- 📖 Add Book

\- 📚 View Books

\- ✏️ Update Book

\- ❌ Delete Book

\- 🔍 Search Book

\- 👤 Add Member

\- 👥 View Members

\- ✏️ Update Member

\- ❌ Delete Member

\- 🔍 Search Member

\- 📕 Issue Book

\- 📗 Return Book

\- 📊 Reports Dashboard

\- 🚪 Logout



\---

&#x20; ▶️ How to Run



1\. Clone this repository.

2\. Open the project in NetBeans IDE.

3\. Configure the Oracle Database connection in `DBConnection.java`.

4\. Run the `LoginFrame.java` file.
